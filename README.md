# IAM Project - Linux Permissions Live Demo

## Platform
Google Cloud Shell

## Overview
Hands-on demo of Linux-based Identity and Access Management (IAM) concepts.

## Topics Covered
- Creating users with `adduser`
- Switching users with `su`
- File permissions with `chmod` (600 and 644)
- Creating groups with `groupadd`
- Adding users to groups with `usermod -aG`
- Verifying access with `id` and `groups`

## File
- `iam-demo-commands.txt` — all commands used in the live demo
